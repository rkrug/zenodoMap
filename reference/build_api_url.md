# Build a Zenodo records API URL

Build a Zenodo records API URL

## Usage

``` r
build_api_url(query, community, size, page)
```

## Arguments

- query:

  Optional search query string.

- community:

  Optional community id.

- size:

  Page size for the request.

- page:

  Page number for the request.

## Value

A fully-qualified URL string.
